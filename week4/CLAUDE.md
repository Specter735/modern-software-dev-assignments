# Panduan Proyek (Week 4 Starter App)
- `backend/`: Berisi aplikasi FastAPI (`main.py`, `routers`, `models`).
- `frontend/`: File UI statis.
- **Testing:** Selalu jalankan `pytest backend/tests/` setelah melakukan perubahan kode.
- **Keamanan:** Jangan pernah menghapus file database `.db` secara permanen.
- **Alur Kerja:** Tulis Test -> Tulis Kode -> Jalankan Test -> Perbaiki jika gagal.
